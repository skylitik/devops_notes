Prerequisities for Node Setup
- Any OS
- Network access from Master 
    * note:Check Firewall rules
- Java, JRE, JDK
- User
- Directory with User ownership
- Tools as required by the Jenkins job (Maven, Ant, Git , etc)

#################################################################
- Create Instance (Slave server, Ubuntu, other SG, key > jenkey)
- Login bash> update && install openjdk-11-jdk -y
- Create user > adduser devops
- Create directory mkdir /opt/jenkins-slave
- Change ownership of directory: chown devops.devops /opt/jenkins-slave -R
- Enable password login ssh: vim /etc/ssh/sshd_config > change Password authentification > yes and del #
- systemctl restart ssh
Jenkins => SSH => Node
